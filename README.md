# google-admanager-api-go

Go SDK for the Google Ad Manager SOAP API. Uses [gowsdl](https://github.com/hooklift/gowsdl) to generate typed service clients.

## Project layout

```
admanager.go       -- Client and config (version-agnostic)
auth.go            -- OAuth2 credential helpers
header.go          -- SOAP RequestHeader
services/
  v202505/
    client.go      -- Version constant + per-service constructors
    network_service/
      network_service.go  -- generated by gowsdl
    order_service/
      order_service.go
    ...
```

## How it works

There are three things you need:

1. **Credentials** -- an `oauth2.TokenSource` (service account, refresh token, or static access token)
2. **Client** -- holds your network code, app name, and authenticated HTTP client
3. **Service** -- a generated SOAP client for a specific API service (e.g. `NetworkService`)

The API version is determined by which `services/vYYYYMM` package you import. There is no version string to pass at runtime.

## Quick start

```go
package main

import (
    "context"
    "fmt"
    "log"

    admanager "google-admanager-api-go"
    v202505 "google-admanager-api-go/services/v202505"
    "google-admanager-api-go/services/v202505/network_service"
)

func main() {
    ctx := context.Background()

    // Step 1: create credentials (pick one)
    ts, err := admanager.ServiceAccountTokenSourceFromFile(ctx, "service-account.json")
    if err != nil {
        log.Fatal(err)
    }

    // Step 2: create the client
    client := admanager.NewClient(ctx, admanager.Config{
        NetworkCode:     "123456",
        ApplicationName: "my-app",
    }, ts)

    // Step 3: create a service and call a method
    networkSvc := v202505.NewNetworkService(client)

    resp, err := networkSvc.GetCurrentNetwork(&network_service.GetCurrentNetwork{})
    if err != nil {
        log.Fatal(err)
    }
    fmt.Println(resp.Rval.DisplayName)
}
```

## Authentication

All helpers return an `oauth2.TokenSource`. Pick whichever fits your setup:

```go
// Service account from JSON key bytes
ts, err := admanager.ServiceAccountTokenSourceFromJSON(ctx, jsonBytes)

// Service account from a key file path
ts, err := admanager.ServiceAccountTokenSourceFromFile(ctx, "path/to/key.json")

// OAuth2 refresh token (for user-granted access)
ts := admanager.TokenSourceFromRefreshToken(ctx, clientID, clientSecret, refreshToken)

// Static access token (you handle refresh yourself)
ts := admanager.StaticTokenSource("ya29.xxxxx")
```

## Creating a service

The version package exports a constructor for every service. Each returns the corresponding generated interface:

```go
import v202505 "google-admanager-api-go/services/v202505"

orderSvc := v202505.NewOrderService(client)
networkSvc := v202505.NewNetworkService(client)
lineItemSvc := v202505.NewLineItemService(client)
```

You still import the individual service package when you need its request/response types:

```go
import "google-admanager-api-go/services/v202505/order_service"

resp, err := orderSvc.GetOrdersByStatement(&order_service.GetOrdersByStatement{
    FilterStatement: &order_service.Statement{Query: "LIMIT 10"},
})
```

Each lives under `services/v202505/<snake_case_name>/`.

## Regenerating services

Edit the `VERSIONS` and `SERVICES` arrays in `scripts/generate.sh`, then run:

```bash
make generate
go mod tidy
```

The `gowsdl` CLI is declared as a [Go tool dependency](https://go.dev/blog/tools) in `go.mod`, so no manual install is needed — `go tool gowsdl` works out of the box after cloning.
